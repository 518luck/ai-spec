- dub 是用 hooks 来请求数据的，每个 hooks 里面通过workid来判断需要求情什么，这个后面还需要详细看hooks的内部结构

| Hook               | 发请求？ | 请求的 API                                  | 触发条件                                  |
| ------------------ | -------- | ------------------------------------------- | ----------------------------------------- |
| useWorkspace       | ✅       | GET /api/workspaces/{slug}                  | 只要能解析出 slug 就发                    |
| useFolders         | ✅       | GET /api/folders?workspaceId=...            | 拿到 workspaceId 后发                     |
| useFolder          | ✅       | GET /api/folders/{folderId}?workspaceId=... | folderId 有值且≠"unsorted" 且非 free 套餐 |
| useLinksCount      | ✅       | GET /api/links/count                        | workspaceId 有值且非 mega workspace       |
| useCurrentFolderId | ❌       | 无                                          | 只读 URL，不发请求                        |
| useAddFolderModal  | ❌       | 无                                          | 只管弹窗显隐                              |
