# 基础信息

## 产品定位
本项目用于管理 AI 使用规范、Prompt 快捷语、Skill 和 Agent 配置。

## 项目结构
- app/：Next.js 路由层，保持薄层
- app/api/：后端入口
- src/：前端业务代码和共享基础设施
- prisma/：数据库 schema 与 migrations

## 默认约束
- 优先使用 TypeScript
- 不直接修改自动生成目录
- 业务实现委托到 src/ 对应层级
- 修改数据库先改 schema，再生成 Prisma Client
